# Default Template Content

This folder contains the editable content for the default Copilot Tasks template.

- `brand.ts`: product name and brand alt text
- `hero.ts`: first-screen headline, description, and background images
- `rolodex.ts`: split-flap sentence text and image sequence
- `features.ts`: feature timeline copy and badges
- `carousel.ts`: carousel images, alt text, headlines, and descriptions
- `cta.ts`: button labels and final CTA headline
- `accessibility.ts`: skip link and section labels
- `meta.ts`: page analytics identifier
- `assets.ts`: shared static asset base path

`../default-template.ts` combines these files into one `LandingPageTemplateConfig`.
