# Userscripts

Browser userscripts installed through Tampermonkey, Violentmonkey, or a compatible userscript manager.

| Path                    | Purpose                                                                                  |
| ----------------------- | ---------------------------------------------------------------------------------------- |
| `bilibili-follow-cycle` | Archived Bilibili follow/unfollow automation panel.                                      |
| `youtube-auto-resume`   | Archived YouTube playback assistant, superseded by `apps/youtube-auto-resume-extension`. |

Each userscript project keeps its own license and installation notes.
